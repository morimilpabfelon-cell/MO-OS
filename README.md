# MO OS

MO OS es una distribución Linux en desarrollo orientada a programación, construcción de software y operación local soberana.

El proyecto comienza con una edición mínima de terminal y evoluciona hacia una distribución instalable con escritorio, herramientas de desarrollo y un dominio Arch Linux coordinado desde una base estable.

> Estado: fundación inicial. No usar todavía para reemplazar el sistema principal de una laptop.
